# No AI Smell Rules

## 目标

把小说从“AI 写得很顺”改成“真人在具体处境里做出反应”。

AI 味通常不是错别字问题，而是：太平均、太正确、太会解释、太像总结、没有现实摩擦。

## 一、禁止空泛情绪

不要写：

```text
他很痛苦。
她的心像被撕裂。
命运终于露出了残酷的一面。
```

改成：

```text
他把止痛药倒出来，又一粒一粒捡回瓶子里。
她夹起的肉停在半空，最后放进了孩子碗里。
他看完群消息，先把电动车钥匙摸出来，又塞回口袋。
```

## 二、禁止假高级词

高频警戒词：

```text
仿佛、似乎、某种、说不清、命运、宿命、光影、破碎、沉默如潮水、眼底翻涌
```

不是完全不能用，而是不能靠它们伪装深度。能用动作就不要用雾。

## 三、禁止旁白替人物洗白

不要写：

```text
她不是坏人，她只是太累了。
他也是真的不容易。
双方都有道理。
```

改成让观众自己看出来：

```text
她把红包退回去时，手指停了很久，最后只退了一半。
他骂完孩子，转身去厨房把最大的一块肉挑出来，没说话。
```

## 四、禁止解释型对话

不要让人物替作者解释设定：

```text
你知道的，我三年前救过你，所以你现在欠我一条命。
```

改成场景里的真实反应：

```text
他把旧病历推过去。
她看见日期，手指一下按住了那一行。
```

## 五、禁止所有人都体面

真人会别扭、会嘴硬、会误解、会算计、会沉默、会讲错话。

更真实的反应：

- 明明心疼，却先骂一句。
- 明明想道歉，却先问吃饭没。
- 明明害怕，却装作嫌麻烦。
- 明明后悔，却只把东西放门口。

## 六、禁止平均节奏

AI 常把每段写得差不多长，每段都像小总结。小说需要节奏差：

- 危机时句子短。
- 信息揭露时留停顿。
- 情绪爆发前减少解释。
- 反转后立刻给具体后果。

## 七、去 AI 味检查表

每写完一段，检查：

```text
有没有具体物件？
有没有具体动作？
有没有现实代价？
人物有没有不完美反应？
这段是否只是在解释？
删掉最后一句总结，意思是否更有力？
```

## 八、重写指令

当用户要求去 AI 味时，优先执行：

1. 删除抽象总结。
2. 删除过度心理描写。
3. 把情绪改成动作。
4. 把立场改成场景。
5. 把解释改成物件。
6. 保留人物的不体面。
7. 让冲突有现实代价。
