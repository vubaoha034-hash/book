# Sample Usage

## 1. 导入书籍

```text
/novel-writing-master 我已经把书放进 sources/books 了，帮我导入并建立技巧库。
```

## 2. 分析某一本

```text
/novel-writing-master 分析《小说技巧》这本书，只看写作技巧，不要总结内容。重点提炼开头钩子、人物发动机、冲突递进和去AI味规则。
```

## 3. 检查我的章节

```text
/novel-writing-master 按技巧库检查下面这一章，打分100分。重点看主线不清晰、矛盾冲突悬浮、节奏慢、AI味、逻辑bug。
```

## 4. 重写开头

```text
/novel-writing-master 把这个开头重写。要求：三秒钩子，十秒留人，冲突落地，少心理描写，去AI味，最后一句让人想继续看。
```

## 5. 生成短小说框架

```text
/novel-writing-master 用我的技巧库，给我10个适合公众号爆款短小说的设定。每个都要有第一句、核心冲突、人物发动机、三次反转、结尾评论点。
```
